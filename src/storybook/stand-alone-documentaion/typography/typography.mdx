## Overview

Like in other UI, typography works by principle of accessibility before aesthetics.
Therefore, the text should be readable and help the user understand whatâ€™s important by well contrasted size and colors hierarchy.

## Fonts

We are using one font for our UI hierarchy: Nunito Sans.
We are using Nunito Sans font for our main titles (heading), and for text, labels and paragraphs.
We don't import the font within our CSS in order to give full control of the fonts which you wish to bring to your client, the following code snippet is what we recommend in order to include our recommended fonts.

```html
<link
  href="https://fonts.googleapis.com/css2?family=Nunito+Sans:ital,wght@0,200;0,300;0,400;0,500;0,600;0,700;1,200;1,300;1,400;1,500;1,600;1,700&display=swap"
  rel="stylesheet"
/>
```

More ever this is how we are building our css variable for our main font families

```css
--font-family: Nunito Sans, Roboto, Noto Sans Hebrew, Noto Kufi Arabic, Noto Sans JP, sans-serif;
--title-font-family: Nunito Sans, Roboto, Noto Sans Hebrew, Noto Kufi Arabic, Noto Sans JP, sans-serif;
```
